
# Insertion Option

Here you store item information. To store info in database you must fill up all option otherwise you can't 
save item informatio. Fill the all empty white fields and press "Add" button to save informtion about item. 
After Successfully done then it show a successull message.


# Edit Option

In this option you can edit item information. First type the item code which one you want to edit then
change the information and just press "Save" button.If you type one item then you fell don't need to edit then
just press "Clear" button to clean all white boxs. After Successfully done then it show a successull message.


# Delete Option


In this option you can delete item. First type the item code which one you want to delete then 
just press the "Delete" button. After Successfully done then it show a successull message.
